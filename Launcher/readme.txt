1. Compile "launcher.c" using cc65.exe with the modified "c64.cfg" to build the launcher front-end to resided at $6000.
2. Assemble "main.asm" using CBM prg studio or equivalent. This simply glues all the programs together into a single file.
3. Put the output "habitat.prg" file on a fresh D64, using C64 DirMaster.
